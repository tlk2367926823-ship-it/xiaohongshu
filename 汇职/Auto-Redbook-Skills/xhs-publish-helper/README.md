# 汇职小红书发布助手

这是一个 Chrome/Edge 扩展，用于把汇职驾校分享活动网页生成的小红书草稿自动填入小红书创作者发布页。

## 安装方式

1. 打开 Chrome 或 Edge。
2. 进入扩展管理页：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 打开「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本目录：`xhs-publish-helper`。

## 使用方式

1. 在汇职活动网页生成分享内容。
2. 点击「自动填入发布界面」。
3. 浏览器会打开小红书创作者发布页。
4. 扩展会尝试自动填入标题和正文。
5. 用户手动上传图片/视频素材。
6. 用户自行检查并点击发布。

## 重要限制

- 该扩展只能辅助填入标题和正文。
- 图片/视频素材仍需用户手动选择上传。
- 扩展不会点击最终发布按钮。
- 小红书页面结构变化时，自动填入可能失效，需要更新选择器策略。
- 移动端浏览器通常不支持此类扩展，因此移动端正式自动发布应走官方 API 或后端代理。
